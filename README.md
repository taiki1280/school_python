# 卒業制作

## コンセプト

* 記憶と思考と判断

## テーマ

* その行動をする前に

## 依頼内容（目的）

* 一目で何を考えているのかがわかるメモアプリを作成する。

* 普段からSNSに投稿をしている人はそのメモを見て本当にその文章が世界に公言できるものなのかを冷静に考えられる。
* 時が経ってに過去の自分が何を思い、どのように生きていたのかを見返す時間に役立てる。
* 自分の思考を客観的に見ることを考える。

### 理想的な利用方法

* 一日一つでもよいので何において、どのように思ったのかをメモする。
* SNSに直接投稿をせず、まずは自分しか見ることのできないメモに書き出してみる。

## 対象者（相手）

* 全日本人あるいは全人類
