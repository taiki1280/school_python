## Elastic Load Balancingの種類

* Application Load Balancer(ALB)
* Network Load Balancer(NLB)
* Classic Load Balancer(CLB)

## 重要語句

* ヘルスチェック
    - 障害検知
* メトリクス
    - 収集した統計情報
* スケールアウト
    - サーバーの数を増やして処理性能を向上させる
* スケールイン
    - サーバーの数を減らす
* スケールアップ
    - サーバーの性能を向上させて、処理性能を向上させる (ex)CPUやメモリの増設など
* AWS IAM
    - ユーザー管理のためのサービス。 (ex)グローバル（１つのアカウントで複数の国を選択できる）
